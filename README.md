# 📊 Finance MIS Dashboard — SQL Project

## 📌 Overview
A comprehensive SQL project simulating real-world Finance MIS operations
including employee management, loan tracking, salary processing,
and department budget analysis — covering Basic to Advanced SQL concepts.

## 🛠️ Tech Stack
- SQL (MySQL)

## 📂 Files
| File | Description |
|------|-------------|
| schema.sql | Database & table creation |
| sample_data.sql | Dummy finance data |
| basic_queries.sql | Basic to medium SQL queries |
| advanced_queries.sql | Advanced SQL concepts |

## 💡 SQL Concepts Covered

### Basic to Medium
- DDL & DML commands
- Primary & Foreign Keys
- JOINs (INNER JOIN)
- WHERE, ORDER BY
- GROUP BY, HAVING
- CASE WHEN
- Aggregate functions (SUM, COUNT, AVG)

### Advanced
- Subqueries
- Views (Virtual MIS Reports)
- Indexes (Performance optimization)
- Stored Procedures
- Window Functions (RANK, ROW_NUMBER, Running Total)

## 🔍 Key Reports Generated
- Department wise salary summary
- Employee loan status tracker
- Budget vs actual spend analysis
- Monthly MIS salary report
- Department wise salary ranking
- Running total of salary payments
